# Day 20 - Sorting Algorithms

## Project

A console-based Sorting Algorithms program that allows users to enter a list of numbers and view the results of three different sorting techniques.

## Features

- Enter multiple numbers separated by spaces
- Bubble Sort
- Selection Sort
- Insertion Sort
- Preserve the original list by sorting a copy
- Menu-driven interface
- Input validation

## Concepts Practiced

- Nested loops
- Lists
- Functions
- Input validation
- Swapping values using a temporary variable
- Copying lists using slicing (`[:]`)
- Bubble Sort
- Selection Sort
- Insertion Sort
- Loop variables (`i` and `j`)
- Algorithm implementation

## What I Learned

- Bubble Sort repeatedly compares adjacent elements and moves the largest value toward the end after each pass.
- Selection Sort repeatedly finds the minimum element from the unsorted portion and places it in its correct position.
- Insertion Sort builds a sorted portion of the list by shifting larger elements and inserting the current value into its correct position.
- Using `original_numbers[:]` creates a copy of the list, allowing each sorting algorithm to run without modifying the original data.
