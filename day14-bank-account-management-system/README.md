# Day 14 - Bank Account Management System

## Project

A console-based Bank Account Management System for creating and managing bank accounts through a menu-driven interface.

## Features

- Create new accounts
- Prevent duplicate account numbers
- View all accounts
- Search accounts by account number
- Deposit money
- Withdraw money with balance validation
- Delete accounts
- View the total balance of all accounts
- Menu-driven interface

## Concepts Practiced

- Functions
- Lists of dictionaries
- Helper functions for code reuse
- Returning objects from functions
- Input validation
- Dictionary manipulation
- Searching data
- Conditional statements
- Reusable validation functions

## What I Learned

- Returning an object from a helper function (`find_account()`) avoids searching the same data multiple times.
- Reusing helper functions makes code cleaner, easier to maintain, and reduces duplication.
- Separating validation logic from business logic improves readability and organization.
