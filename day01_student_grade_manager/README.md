# Day 1 - Student Grade Manager

## Features

- Store student names and marks
- Validate input
- Calculate average marks
- Find highest and lowest scorer
- Assign grades

## What I learned

- How to use lists and dictionaries together.
- The difference between iterating using an index and iterating over elements directly.
- Variables in Python hold references to objects, which helped me understand why modifying a dictionary affects all references to it.
