# Day 1 - Student Grade Manager

## Project

A console-based student grader manager which assigns grade according to the input marks.

## Features

- Store student names and marks
- Validate input
- Calculate average marks
- Find highest and lowest scorer
- Assign grades

## What I Learned

- How to use lists and dictionaries together.
- The difference between iterating using an index and iterating over elements directly.
- Variables in Python hold references to objects, which helped me understand how modifying a dictionary through one reference changes the same dictionary stored in a list.

## Concepts Practiced

- Variables
- Lists
- Dictionaries
- Loops (`for`, `while`)
- Input validation
- Conditional statements
- f-strings
- Iterating over lists

## Future Improvements

- Allow updating a student's marks.
- Delete a student record.
