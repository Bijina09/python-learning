# Day 3 - Contact Book

## Project

A console-based contact management program that allows users to store, search, view, and delete contacts.

## Features

- Add a new contact
- Prevent duplicate contact names (case-insensitive)
- View all contacts
- Search contacts by name
- Delete contacts
- Menu-driven interface

## Concepts Practiced

- Functions
- Lists
- Dictionaries
- Loops (`for`, `while`)
- Conditional statements (`if`)
- `return`
- `break`
- `match-case`
- `enumerate()`
- `list.remove()`
- Case-insensitive string comparison using `lower()`

## What I Learned

- How to organize a larger program into multiple functions.
- How to remove an item from a list using `list.remove()`.
- The difference between using `break` to exit a loop and `return` to exit a function.
- Variables in Python store references to objects, not the objects themselves.
- Reusing the same variable name can accidentally change what it references, leading to unexpected bugs.
