# Day 13 - Contact Book Version 2

## 📌 Project Overview

A console-based Contact Book application built with Python. This improved version allows users to manage contacts while introducing favorite contacts, better validation, duplicate checking during updates, and reusable helper functions.

## ✨ Features

- Add new contacts
- View all contacts
- Search contacts by name
- Update contact details
- Delete contacts
- Toggle favorite status
- View favorite contacts only
- Prevent duplicate contact names
- Validate integer input for phone numbers
- Validate Y/N input for favorite status
- Prevent empty inputs
- Clear screen between operations

## 🛠 Concepts Practiced

- CRUD Operations
- Lists of Dictionaries
- Helper Functions
- Input Validation
- Boolean Values
- Function Reusability
- Duplicate Checking
- Object Identity (`is`)
- `continue` Statement
- Menu-Driven Programming

## 📚 What I Learned

- Reusable helper functions make the code cleaner and reduce duplication.
- Boolean values are useful for implementing toggle features.
- Using `is` compares object identity, allowing duplicate checking to ignore the contact currently being edited.
- `continue` skips only the current iteration of a loop.
- Breaking larger problems into helper functions makes programs easier to maintain.
