# Day 19 - Pattern Printing

## Project

A console-based Pattern Printing program that demonstrates how to generate different star (`*`) patterns using nested loops and a menu-driven interface.

## Features

- Print a left-sided triangle
- Print an upside-down triangle
- Print a centered triangle (pyramid)
- Menu-driven interface
- Input validation for menu selection

## Concepts Practiced

- Nested loops
- `for` loops with `range()`
- Printing with `end=""`
- Pattern generation
- Loop control
- Functions
- Menu-driven programming
- Input validation

## What I Learned

- Nested loops can be used to control both rows and columns when generating patterns.
- Centered patterns require printing spaces before stars to achieve proper alignment.
- The number of stars in each row follows the pattern `2 * row + 1`.
- Breaking a problem into smaller functions makes the program easier to organize and understand.

## Future Improvements

- Add right-sided triangle patterns.
- Add diamond and hollow triangle patterns.
- Allow the user to choose the height of the pattern.
- Add number and alphabet patterns.
- Print patterns using different symbols instead of only `*`.
